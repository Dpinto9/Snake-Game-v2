# Snake Game v2

A classic Snake game built entirely with vanilla JavaScript, HTML, and CSS. No frameworks, no libraries.

## Features

- Smooth snake movement with keyboard controls
- Collision detection for walls and self
- Score tracking that increases with each food item consumed
- Game over and restart flow
- Custom assets and styling for a polished visual experience

## Tech Stack

- **Language:** JavaScript (vanilla)
- **Markup:** HTML5 Canvas / DOM
- **Styling:** CSS3

## Project Structure
```
├── Assets/         # Game sprites and visual assets
├── Scripts/        # Game logic (movement, collision, scoring)
├── index.html      # Entry point
└── style.css       # Styling and layout
```

## How to Play

1. Clone the repo:
```bash
git clone https://github.com/Dpinto9/Snake-Game-v2.git
```
2. Open `index.html` in your browser
3. Use the arrow keys to control the snake
4. Eat food to grow and increase your score
5. Avoid hitting the walls or yourself
